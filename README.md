# 🌍 Smart Ground Water Level Prediction

This project is based on predicting groundwater levels using Machine Learning and IoT.

## 🚀 Features
- Predict groundwater level
- Uses Machine Learning model
- IoT integration for real-time data

## 🛠️ Technologies Used
- Python
- Machine Learning
- IoT
- Pandas, NumPy, Scikit-learn

## 📊 Dataset
Dataset is not uploaded due to large size.

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the project:
   python main.py
